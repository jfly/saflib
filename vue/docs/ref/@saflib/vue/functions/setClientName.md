# Function: setClientName()

> **setClientName**(`client`): `void`

Call when the SPA starts, providing the name of the client. It should be the same as the package name, without the org prefix, so "web-auth" or "web-landing".

## Parameters

| Parameter | Type     |
| --------- | -------- |
| `client`  | `string` |

## Returns

`void`
