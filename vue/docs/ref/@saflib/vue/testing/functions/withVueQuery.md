# Function: withVueQuery()

> **withVueQuery**\<`T`\>(`composable`, `queryClient?`): \[`T`, `App`\<`Element`\>, `any`\]

```typescript
const [query, app, queryClient] = withVueQuery(() =>
  useQuery(getContact(contactId)),
);
await query.refetch();
expect(query.data.value).toEqual(mockContact);
```
