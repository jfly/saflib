# Function: continueWorkflow()

> **continueWorkflow**(`actor`): `void`

Convenience function to continue a workflow which has halted because a prompt was shown. Signals every active actor to "continue".

## Parameters

| Parameter | Type |
| ------ | ------ |
| `actor` | `AnyActor` |

`void`
